# Terminal Configurations

Personal terminal emulator configs for Windows and macOS.

## Included

- WezTerm (Windows): `wezterm/.wezterm.lua`
- Ghostty (macOS): `ghostty/config`

## Ghostty Installation

Recommend using the tip (nightly) version for more features like `cmd+f` search:

```bash
brew install --cask ghostty@tip
```

For stable version:

```bash
brew install --cask ghostty
```
